🐦 Flappy Bird

Este projeto consiste no desenvolvimento de uma recriação do clássico jogo Flappy Bird, implementado inteiramente utilizando a linguagem de programação Java. O objetivo principal foi aplicar conceitos fundamentais e intermediários de programação, lógica e estrutura de jogos em um projeto prático e interativo.

⸻

🎮 Sobre o Projeto

O jogo segue a proposta original do Flappy Bird: o jogador controla um pássaro que deve atravessar obstáculos (canos) sem colidir, acumulando pontos ao longo do percurso.

Durante o desenvolvimento, foram trabalhados diversos aspectos importantes da programação, como:

* Lógica de movimentação e física básica (gravidade e impulso)
* Detecção de colisão
* Controle de eventos (teclado)
* Atualização contínua da tela (game loop)
* Manipulação de gráficos em Java

⸻

🛠️ Tecnologias Utilizadas

* Java (linguagem principal)
* Bibliotecas gráficas nativas (como Swing ou JavaFX, dependendo da sua implementação)
* Programação orientada a objetos (POO)

⸻

⚙️ Funcionalidades

* Controle do personagem com interação do usuário
* Sistema de pontuação em tempo real
* Geração dinâmica de obstáculos
* Detecção de colisões precisa
* Reinício do jogo após derrota
* Interface gráfica simples e funcional

⸻

🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

* Praticar conceitos de Programação Orientada a Objetos (POO)
* Compreender a estrutura de um jogo 2D
* Aprimorar habilidades em lógica de programação
* Trabalhar com interfaces gráficas no Java
* Criar um projeto completo para portfólio

⸻

🚀 Aprendizados

Ao desenvolver este jogo, foram consolidados conhecimentos importantes como:

* Organização de código em classes e métodos
* Estruturação de um loop de jogo eficiente
* Manipulação de eventos do usuário
* Controle de estados dentro de uma aplicação
* Boas práticas de desenvolvimento em Java
